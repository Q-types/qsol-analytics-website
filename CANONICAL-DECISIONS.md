## 4. Methodology

### The 5-Step Process (Standardised)

| Step | Name | Duration | Description |
|------|------|----------|-------------|
| 1 | Discovery | 30 min call | Discuss setup, pain points, goals |
| 2 | Assessment | 1-3 days | Review data, systems, workflows |
| 3 | Build | 2-4 weeks | Working prototype with real data |
| 4 | Refine | 1 week | Adjustments based on feedback |
| 5 | Deploy & Support | Varies | Documentation, handover, optional support |

### Key Differentiators

1. **Rapid Time-to-Value** — Prototype in 2-4 weeks
2. **Signal Over Noise** — Focus on what matters
3. **Measurable Outcomes** — Success metrics agreed upfront
4. **Right-Sized Solutions** — Won't oversell
5. **AI Where It Helps** — ML only when it adds genuine value
6. **Founder Access** — David on every project

## 7. Compliance & Security

### Data Handling Principles

1. **Access:** Only access data needed for specific project
2. **Storage:** Data stays on client systems where possible
3. **Transfer:** Encrypted (HTTPS/TLS) always
4. **People:** Only project team (typically David) accesses data
5. **Deletion:** Project data deleted after completion unless retention requested

### GDPR Position
- Operate as data processor when handling client data
- Will sign DPA if required
- No personal data collection beyond project needs
- No data selling or sharing

### Audit Support
- Clear documentation of data sources, transformations, logic
- Version-controlled code
- Outputs traceable to source data
- Plain-English explanations of models

## 8. Content Standards

### Writing Style

| Element | Standard |
|---------|----------|
| Spelling | British English (realise, colour, organisation) |
| Currency | GBP (£) |
| Numbers | Use numerals for statistics; words for small counts |
| Abbreviations | Define on first use (e.g., "SME (small and medium enterprise)") |
| Jargon | Always explain in parentheses or "What this means" section |

### Page Structure

Every service page includes:
1. Hero (H1, subhead, CTA)
2. Problem section
3. Solution/Approach section
4. Deliverables section
5. Process section (5 steps)
6. Results/outcomes section
7. Pricing guidance
8. CTA section
9. Related content links
10. Structured data (JSON-LD)

### Meta Data Standards

| Element | Length | Content |
|---------|--------|---------|
| Title tag | 50-60 chars | Primary keyword + brand |
| Meta description | 150-160 chars | Action-oriented, includes keyword |
| H1 | One per page | Contains primary keyword |
