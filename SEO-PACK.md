## 3. Technical SEO Requirements

### URL Structure

| Page Type | Pattern | Example |
|-----------|---------|---------|
| Homepage | / | qsol-analytics.co.uk/ |
| Service | /services/{slug}/ | /services/automation-workflow/ |
| Case Study | /case-studies/{slug}/ | /case-studies/logistics-automation-demo/ |
| Blog | /insights/{slug}/ | /insights/automation-small-business-uk/ |
| Legal | /{slug}/ | /privacy-policy/ |

### Required Technical Elements

| Element | Requirement |
|---------|-------------|
| SSL | HTTPS required |
| Mobile responsive | Must pass Google mobile-friendly test |
| Page speed | Target < 3s load time |
| Sitemap | XML sitemap at /sitemap.xml |
| Robots.txt | Allow all, reference sitemap |
| Canonical URLs | Self-referencing on all pages |
| 404 page | Custom branded page |
| Redirects | 301 redirects for any URL changes |

### Structured Data (Schema.org)

| Page Type | Schema Type |
|-----------|-------------|
| Homepage | Organization |
| Service pages | Service |
| Case studies | Article |
| Blog posts | Article |
| FAQ | FAQPage |
| Contact | ContactPoint (in Organization) |

## 4. Internal Linking Plan

### Hub-and-Spoke Structure

```
Homepage (Hub)
├── Services Overview (Sub-hub)
│   ├── Automation Service → Blog: Automation → Case Study: Logistics
│   ├── Dashboards Service → Blog: Dashboards → Case Study: Manufacturing
│   ├── Segmentation Service → Blog: Segmentation → Case Study: Retail
│   └── Forecasting Service → Blog: Forecasting/Anomaly → Case Study: Distribution
├── About → FAQ
├── Case Studies Index
│   ├── Logistics Automation
│   ├── Retail Segmentation
│   ├── Manufacturing Dashboard
│   └── Distribution Forecasting
└── Blog Index
    ├── Automation Small Business
    ├── Data Dashboards SMEs
    ├── Customer Segmentation Examples
    ├── Time Series Forecasting
    └── Anomaly Detection
```

### Cross-Linking Requirements

| From | To | Anchor Text |
|------|----|-------------|
| Each service page | Related blog article | "Read more: [Topic]" |
| Each service page | Related case study | "See how it works: [Case Study]" |
| Each blog article | Related service page | "Our [Service] service" |
| Each case study | Related service page | "Learn more about [Service]" |
| Homepage | All service pages | Service card links |
| Homepage | Featured case studies | "See how it works" |
| FAQ | Service pages | In answer text |

### Contextual Links in Body Copy

Add 2-3 contextual links per page within body copy (not just navigation):

| Page | Suggested Links |
|------|-----------------|
| Homepage | Link to FAQ from "Common Questions" section |
| Automation Service | Link to forecasting when mentioning scheduled data |
| Dashboards Service | Link to automation when mentioning data pipelines |
| Segmentation Service | Link to forecasting for churn prediction |
| Forecasting Service | Link to dashboards for visualisation |
